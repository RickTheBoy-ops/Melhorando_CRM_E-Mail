Sistema de Gerenciamento de Email

Token de Acesso:

Use este token no header Authorization como: Bearer SEU_TOKEN

🚀 Endpoints Disponíveis:

Gateway: http://localhost:8080

Rotas Públicas:

  • GET /health - Status do sistema
  • POST /api/v1/auth/login - Login
  • POST /api/v1/auth/register - Registro

Rotas Protegidas: Requerem token de autorização

  • /api/v1/email/* - Gerenciamento de emails
  • /api/v1/contacts/* - Gerenciamento de contatos
  • /api/v1/campaigns/* - Campanhas de marketing
